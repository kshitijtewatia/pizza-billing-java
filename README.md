# Pizza Billing System (Java)

A simple console-based pizza billing project built using Java and OOP concepts.

## Features
- Veg & Non-Veg pizza selection
- Extra cheese and toppings
- Takeaway option
- Final bill calculation

## Concepts Used
- Classes and Objects
- Constructors
- Encapsulation
- Conditional logic
- Scanner input

## How to Run
1. Compile:
   javac src/*.java
2. Run:
   java Main

## Future Improvements
- Multiple pizza orders
- Bill breakdown
- Discount coupons
- Menu-driven system
